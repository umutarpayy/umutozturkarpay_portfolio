Automated Learning Objective Classification System

Project Summary

This advanced text classification project automatically maps question texts to corresponding Turkish Ministry of Education (MEB) learning objectives. Using FastText for Multi-Label Text Classification, the system processes educational content with high accuracy. The project implements separate FastText models for each grade level and subject, trained on approximately 50,000 original data points supplemented with synthetic data. The system achieves remarkable accuracy rates, with over 99% accuracy for 50% of questions using original data, and 55-56% accuracy with synthetic data. Advanced data augmentation techniques including back-translation and paraphrasing were employed to enhance the training dataset.

Technologies Used:

  • FastText
  • GPT-3.5 and LLaMA
  • Python NLP Libraries
  • Data Augmentation Tools
  • Machine Learning Frameworks

Project Features:

  • Multi-Label Text Classification
  • Synthetic Data Generation
  • High-Accuracy Prediction
  • Grade-Level Specific Models
  • Automated Learning Objective Mapping
  • Large-Scale Data Processing
  • Advanced Text Analysis
