Project Summary

The purpose of this project is to enable interaction with a PDF uploaded to the website, allowing for conversational engagement.

Technologies Used:

  • Flask
  • Django
  • RAG
  • gpt-3.5-turbo
  • Claude Sonnet

Project Features:

  • PDF File Upload and Processing
  • Natural Language Processing and Understanding
  • Smart Question-Answer System
  • Multi-Language Support
  • Real-Time Chat Interface
  • Context-Aware Responses
  • Customizable AI Model
