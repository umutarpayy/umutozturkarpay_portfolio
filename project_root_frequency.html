Root Frequency Analysis

Turkish Educational Content Root Word Analysis System

Project Summary

This project analyzes the root word frequency statistics of question contents obtained through TPNG, categorized by educational level and grade. Using Zemberek, a natural language processing tool for Turkish, the project extracts root words from questions and performs statistical analysis. The system processes extensive question data to identify the most commonly used root words, providing valuable insights into vocabulary patterns across different educational levels. This analysis helps understand the linguistic complexity and vocabulary distribution in educational content, which can be particularly useful for curriculum development and content optimization.

Technologies Used:

  • Python
  • Zemberek NLP
  • Data Analysis Tools
  • Statistical Processing
  • Data Visualization Libraries

Project Features:

  • Turkish Root Word Extraction
  • Statistical Analysis
  • Level-based Classification
  • Frequency Visualization
  • Educational Content Analysis
  • Grade-specific Word Distribution
  • Comparative Analysis Tools
