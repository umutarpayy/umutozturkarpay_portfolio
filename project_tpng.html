TPNG

Automated Question Bank Processing System with Advanced Computer Vision

Project Summary

TPNG is a sophisticated application designed to automatically process question bank books and convert them into structured XML files. Using YOLO (You Only Look Once) for object detection, the system can identify and locate 16 different classes of elements within textbook pages, including question boundaries, options, and explanations. The project integrates multiple computer vision technologies, including OCR (Optical Character Recognition) tools like PyTesseract and EasyOCR, to accurately extract and organize question content into properly sequenced tests.

Technologies Used:

  • YOLO (Object Detection)
  • PyTesseract & EasyOCR & OCR Space
  • OpenCV
  • Python
  • XML Processing

Project Features:

  • Automated Question Detection
  • Multi-Class Object Recognition
  • Text Extraction and Processing
  • Structured XML Output
  • Test Organization System
  • High Accuracy Detection
  • Batch Processing Capability
